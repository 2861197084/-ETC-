# ETC 数据目录

此目录存放 CSV 数据文件。

## 目录结构

```
data/
├── expanded/           # CSV 数据文件
│   ├── 2023-12-01.csv  # 历史数据 (2023年12月)
│   ├── 2023-12-02.csv
│   ├── ...
│   ├── 2024-01-01.csv  # 实时模拟数据 (2024年1月)
│   └── 2024-01-31.csv
```

## 数据格式

CSV 字段：
- `GCXH` - 过车序号
- `XZQHMC` - 行政区划名称
- `KKMC` - 卡口名称
- `FXLX` - 方向类型 (进城/出城)
- `GCSJ` - 过车时间 (示例：`2023/12/1 0:00:01`)
- `HPZL` - 号牌种类
- `HP` - 号牌号码
- `CLPPXH` - 车辆品牌型号
